# Ali Ahsan - Portfolio Website

A personal portfolio website showcasing projects, experience, and certifications in Data Science and AI Engineering. Designed with a clean, terminal-inspired aesthetic focusing on performance and minimalism.

## Tech Stack
* React
* Vite
* CSS (Custom properties, animations)

## Local Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Deployment
This project is configured to be deployed easily on Vercel. Connect the GitHub repository and Vercel will automatically detect the Vite build settings.
